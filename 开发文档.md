# 知识库展示网站开发文档

**项目名称：** 知识库文档系统
**文档版本：** v1.0
**编写日期：** 2026-05-29
**文档状态：** 待确认

---

## 一、项目概述

基于原型图，开发一个现代化的知识库展示网站，采用经典的**三栏式布局**：
- **左侧栏**：可折叠的多级分类导航树
- **中间栏**：文章详细内容展示区域
- **右侧栏**：当前文章的章节导航（On This Page）

---

## 二、功能需求分析

### 2.1 左侧导航分类系统

#### 功能说明
左侧边栏展示知识库的多级分类结构，支持无限层级的分类嵌套。

#### 分类结构示例
```
📚 入门指南
   ├─ 简介
   ├─ 认证方式
   ├─ 请求频率限制
   └─ 错误码说明

👥 用户
   ├─ GET /v2/users - 获取用户列表
   ├─ GET /v2/users/:id - 获取单个用户
   ├─ POST /v2/users - 创建用户
   ├─ PUT /v2/users/:id - 更新用户
   └─ DELETE /v2/users/:id - 删除用户

📁 项目
   ├─ 获取项目列表
   └─ 创建项目

🔗 Webhook
   ├─ 获取Webhook列表
   └─ 创建Webhook

💻 SDK
   ├─ JavaScript
   ├─ Python
   ├─ Go
   └─ Rust
```

#### 交互特性
- ✅ 支持分类的**展开/折叠**操作
- ✅ 分类标题可以设置图标
- ✅ 文章链接支持HTTP方法标签（GET、POST、PUT、DELETE）
- ✅ 当前选中的文章高亮显示（左侧显示蓝色竖条）
- ✅ 鼠标悬停时显示浅色背景
- ✅ 侧边栏可滚动，支持滚动到底部自动加载更多

#### 技术要点
- 使用树形数据结构存储分类信息
- 支持懒加载子分类（可选优化）
- 记住用户展开/折叠的状态（localStorage）

### 2.2 文章详情展示区域

#### 功能说明
右侧主内容区展示选中文章的完整内容，支持丰富的文档元素。

| **段落文本** | 支持富文本渲染 | 普通说明文字 |
| **代码块** | 多语言语法高亮 | ```javascript ``` |

### 2.3 文章导航（On This Page）

#### 功能说明
右侧固定栏显示当前文章的所有章节，支持快速跳转。

#### 交互特性
- ✅ 列出文章中所有 H2、H3 标题
- ✅ 点击跳转到对应章节（平滑滚动）
- ✅ 滚动时自动高亮当前阅读位置
- ✅ 仅在屏幕宽度 >= 1200px 时显示
- ✅ 固定在视口右侧，不随页面滚动

#### 视觉样式
```
On This Page
─────────────
• Query Parameters        ← H2 标题
  ◦ 参数说明               ← H3 子标题
  ◦ 参数约束               ← H3 子标题
• Example Request         ← H2 标题
  ◦ cURL 示例
  ◦ JavaScript 示例
• Try It ⚡
• Response Fields
• Response Codes
```

## 四、技术架构建议

### 4.1 技术栈选择

#### 方案一：纯前端方案（推荐）
**适用于：** 静态文档、博客、API文档

| 技术 | 用途 | 优势 |
|------|------|------|
| **Vite + Vue 3** | 构建工具 + 框架 | 快速开发、响应式、性能优秀 |
| **Vue Router** | 路由管理 | SPA路由支持、懒加载 |
| **Pinia** | 状态管理 | 轻量、TypeScript支持 |
| **VuePress / VitePress** | 文档框架 | 开箱即用、Markdown支持 |

**推荐理由：**
- 开发效率高，社区活跃
- Vue 3 Composition API 灵活
- 生态完善，插件丰富
- 可以复用原型图的样式代码

#### 方案二：Next.js 方案
**适用于：** 需要SEO、SSR的文档站

| 技术 | 用途 | 优势 |
|------|------|------|
| **Next.js 14** | React框架 | App Router、SSR/SSG |
| **React** | UI框架 | 生态庞大 |
| **Tailwind CSS** | 样式框架 | 快速开发 |

**推荐理由：**
- 优秀的SEO支持
- SSR/SSG灵活切换
- React生态丰富

#### 方案三：传统HTML/CSS/JS
**适用于：** 简单文档、快速原型

**推荐理由：**
- 学习成本低
- 原型图可直接复用
- 无需构建工具

## 五、页面交互流程

### 5.1 路由设计
```
/                           # 首页（可配置）
/docs                       # 文档首页
/docs/:category             # 分类页（可选）
/docs/:category/:article    # 文章详情页
/search?q=keyword           # 搜索结果页
```

### 5.2 用户操作流程

#### 流程1：浏览文档
```
1. 用户打开网站 → 显示首页或文档列表
2. 点击左侧分类 → 展开子分类
3. 点击文章标题 → 右侧显示文章内容
4. 滚动页面 → 右侧导航自动更新当前章节
5. 点击右侧导航 → 平滑滚动到对应章节
```

#### 流程2：搜索文档
```
1. 点击搜索框 或 按 ⌘K
2. 输入关键词 → 实时显示搜索结果
3. 点击搜索结果 → 跳转到文章对应位置
```

#### 流程3：切换主题
```
1. 点击主题图标
2. 选择主题（浅色/深色/跟随系统）
3. 页面平滑过渡到新主题
4. 保存偏好到 localStorage
```

## 八、关键实现要点

### 8.1 滚动监听（Scroll Spy）
```javascript
// 使用 Intersection Observer 实现
const observer = new IntersectionObserver(
  (entries) => {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        // 更新当前高亮的章节
        currentSection.value = entry.target.id;
      }
    });
  },
  {
    rootMargin: '-20% 0px -80% 0px',
    threshold: 0
  }
);

// 监听所有标题元素
document.querySelectorAll('h2[id], h3[id]').forEach(heading => {
  observer.observe(heading);
});
```

### 8.2 平滑滚动
```javascript
const scrollToSection = (id) => {
  const element = document.getElementById(id);
  if (element) {
    element.scrollIntoView({
      behavior: 'smooth',
      block: 'start'
    });
  }
};
```

### 8.3 分类状态持久化
```javascript
// 保存展开状态到 localStorage
const saveExpandedState = () => {
  const expandedIds = expandedCategories.value;
  localStorage.setItem('sidebar-expanded', JSON.stringify(expandedIds));
};

// 页面加载时恢复状态
const loadExpandedState = () => {
  const saved = localStorage.getItem('sidebar-expanded');
  if (saved) {
    expandedCategories.value = JSON.parse(saved);
  }
};
```

## 九、注意事项

1. **原型图复用**：原型图的HTML/CSS代码可以直接复用，只需转换为Vue组件
2. **代码高亮**：推荐使用 Prism.js 或 Highlight.js 实现代码语法高亮
3. **Markdown支持**：可考虑使用 Markdown 解析器管理文章内容

7. **性能**：注意懒加载和代码分割，避免首屏加载过慢
